# AiDeNote 录音笔记技能

智能录音笔记管理助手 - 通过 OpenClaw 小龙虾快速查找和管理你的 AiDeNote 录音笔记。

## 功能特性

- 📝 查询和搜索录音笔记
- 🎯 获取完整转写文本和 AI 总结
- 🗂️ 知识库语义检索
- 📁 文件夹分类管理

## 安装方法

复制以下内容发送给你的 OpenClaw 小龙虾：

```
请安装 AiDeNote 录音笔记技能，帮我管理和查找录音笔记。
技能地址：https://clawhub.ai/ajingmiao/aidenote-skill
```

## 配置说明

1. 访问 [API Key 管理页面](https://www.aidenote.cn/apikey)
2. 输入密钥名称（如：OpenClaw小龙虾）
3. 点击"生成访问密钥"
4. 复制生成的 API Key
5. 按照小龙虾提示配置 API Key

## 使用示例

- "帮我看看最近的录音笔记"
- "找一下关于项目会议的笔记"
- "在知识库里搜索产品需求"
